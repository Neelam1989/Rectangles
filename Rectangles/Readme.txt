# The RectanglesCommands solutions

1. There are 3 commands for rectangles along with their X and y cordinates
a. "PLACE" (to place an rectangle on grid).
b. "FIND" (to find rectangle that is placed already).
c. "Remove" (to remove the rectangle from the grid)

Steps to run application

1. first application will promot for grid x and y cordinates (user needs to enter valid grid cordinates here).
2. enter Place commands to place rectangle on grid than enter rectangle valid cordinates.
3. user can enter any commonds "Find" or "Remove" followed by its cordinates.
4. if rectangle is removed from grid it will not be find on grid.
